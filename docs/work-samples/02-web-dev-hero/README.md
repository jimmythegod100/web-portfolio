# Responsive Hero Section — Front-End Sample

**Agency:** Vincere Media Works  
**Role:** Senior Front-End Developer  
**Deliverable:** Clean HTML + CSS hero (flexbox, mobile-first)

## Files

- `index.html` — semantic structure: header, hero media placeholder, headline, lede, dual CTAs  
- `styles.css` — design tokens, flex layout, responsive breakpoints, reduced-motion support  

## Preview

Open `index.html` in a browser (or serve the folder locally).

```bash
cd docs/work-samples/02-web-dev-hero
python3 -m http.server 8765
# → http://localhost:8765
```

## Highlights

- Full-viewport hero with background image placeholder via CSS custom property  
- Bold Fraunces headline + Source Sans 3 body  
- Primary coral CTA + ghost secondary action  
- Flexbox content stacking; desktop centers content vertically  
- Accessible focus styles and `prefers-reduced-motion` handling
